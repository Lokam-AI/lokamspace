
## ⚙️ Getting Started

```bash
# Clone the repo
git clone https://github.com/Lokam-AI/lokamspace.git
cd lokamspace

# Setup virtual environment
python -m venv venv
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Run agent (e.g., SAMBA)
cd agents/samba
python main.py
